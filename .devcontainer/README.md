## 文件清单

文件名 | 说明
---|---
`devcontainer.json` | vscode devcontainer 配置文件
`Dockerfile.base` | 基础镜像
`Dockerfile` | 运行时镜像，主要是节省编译时间
`docker-compose.yml` | 项目服务依赖
`create-db-user.sql` | 用于初始化 Postgres 默认用户及权限
